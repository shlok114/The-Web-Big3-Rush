# The Web Big3 Rush

A 10-day intensive practice focused on mastering the core web technologies:
HTML, CSS, and JavaScript.

## Purpose

To prepare for a 2-hour website designing competition by:
- Refurbishing HTML and CSS
- Implementing simple JavaScript features

## Structure

Each folder represents one focused learning day:

- Day 01 – HTML Structure
- Day 02 – CSS Layout (Flexbox)
- Day 03 – Responsive Design
- Day 04 – UI Polish
- Day 05 – JavaScript Basics
- Day 06 – Interactive Feature
- Day 07 – Form Validation
- Day 08 – Mock Build 1
- Day 09 – Mock Build 2
- Day 10 – Final Simulation

## Story
- In the last year of my diploma (2nd sem) I've learnt HTML and CSS as a part of curriculum. This is a light revision of the same.
- Moreover, as JS wasn't primarily in the syllabus, the focus will be to use simple and effective JS in our websites

## Will this help you?
- Yes, If you also want to refresh the frontend basics
- Implement effective JS in webpage
